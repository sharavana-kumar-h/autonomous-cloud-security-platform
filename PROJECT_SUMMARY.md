# Project Summary

## One-Line Pitch

Built a cloud security and observability platform backend that ingests telemetry from databases, Kubernetes, and applications, clusters behavior into sessions, produces ATT&CK-style detections, and recommends automated mitigations.

## Resume Version

- Designed a FastAPI-based cloud security analytics platform with normalized telemetry ingestion, database wire-protocol style parsing, and a durable replayable event log
- Implemented behavioral session clustering, anomaly scoring, MITRE ATT&CK mapping, guarded remediation planning, selector-based tenant policy enforcement, persisted control-plane policy CRUD APIs, time-bounded exception workflows, staged multi-approver remediation approvals with revocation and approver-group routing, tenant, namespace, environment, workload, service-account, and label-aware JWT controls, and admin user lifecycle management
- Added sample payloads, tests, containerization, and Render deployment scaffolding for a production-leaning portfolio project
